• В JS есть конструкция которая позволяет выводить что-то в консоль(console.log).
  • В JS есть три скалярных типа данных: number, string, boolean
  • В JS есть операторы разных типов(перечислены в main.js)
  • В JS есть ключевое слово let для объявления переменных. После let следует имя переменной.
  • В JS есть ключевые слова while и for для объявления цикла.
  • В JS есть ключевое слово if для объявления условия.
